Hero slideshow images (optional local files)
==========================================

The home hero uses a slideshow. By default it loads high-quality
exterior / campus photos from Unsplash (see index.php).

To use your own Sant Manjit Singh College photography:

1. Add 3–4 JPG or WebP files here, e.g.:
   - hero-campus-01.jpg  (main building / entrance)
   - hero-campus-02.jpg  (another exterior angle)
   - hero-campus-03.jpg  (grounds / signage)
   - hero-campus-04.jpg  (optional)

2. In config.php define SMS_HERO_IMAGES as a PHP array of paths
   (see comment block in config.php after you add the constant),
   or edit the $sms_hero_slides array in index.php.

Recommended: 2000–2400px wide, landscape, compressed for web.
